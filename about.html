The Project

The Creature

The Eclipse Fun Station is a project developed for the NASA SPACE APP HACKATHON 2023. It is based on the challenge "Eclipses: Perspective is Everything" This challenge requested participants to create an activity, game, or story explaining the mechanics of an eclipse.


For this project we created a memory game with the different types of eclipses and we included an additional page so people can read about how eclipses occur and more. The project was developed with HTML, CSS and Javascript and tries to offer a simple but fun online activity page for kids and adults to learn about eclipses.
